# Swaraamrita – Full Stack Web Platform

A web platform designed for music lovers to explore ragas, playlists, and music theory.

## Features
- User authentication (login/register)
- Playlist management
- Search functionality
- Responsive UI

## Tech Stack
- Frontend: HTML, CSS, JavaScript, ReactJS
- Backend: Java
- Database: MySQL
- APIs: RESTful APIs
